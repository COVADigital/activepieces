## How to use
1. Select CSV as an action
2. There are two action options "Convert JSON to CSV" and "Convert CSV to JSON"
3. Depending on the step 2 above, enter the "JSON input" or "CSV Text"
4. Choose "CSV contains headers" if the text or the CSV generated should have headers.
